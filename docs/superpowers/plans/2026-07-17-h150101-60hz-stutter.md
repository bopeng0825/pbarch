# H150101 60 Hz Stutter Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Keep PicoDrive at NTSC 60 Hz on H150101 while protecting continuous audio with isolated automatic frame skips and reducing SDL2 RGB565 hot-path copying.

**Architecture:** PicoDrive will use H150101-only override defaults for automatic frame skipping, while explicit saved configuration remains authoritative. SDL2 profiling will expose audio underruns and minimum buffer occupancy, and the hardware-scaled RGB565 path will allocate/copy a shadow frame only when HUD compositing needs it; menu entry will use a one-time renderer readback.

**Tech Stack:** C99, libretro, SDL2, PicoDrive core options, GNU Make

## Global Constraints

- Preserve NTSC `60.000000` Hz emulation speed.
- Preserve the user's existing uncommitted `plat_sdl.c` change at `spec.channels`.
- Do not change defaults on platforms other than H150101.
- Do not add or run repository tests; the user will validate on the H150101 device.
- Do not modify bundled core source directly; use `overrides/` and existing downstream patches.
- Do not add double-buffered textures in this change.

- [ ] **Step 2: Count each callback that emits silence**

In `plat_sound_callback()`, set a local flag only when the active audio callback
must enter the zero-fill loop. Increment the SDL atomic once per affected
callback, not once per silent sample:

```c
static void plat_sound_callback(void *unused, uint8_t *stream, int len)
{
	int16_t *p = (int16_t *)stream;
#ifdef USE_SDL2
	bool underrun = false;
#endif

	/* existing paused/empty-buffer early return and buffered-copy loop */

	while (len > 0) {
#ifdef USE_SDL2
		underrun = true;
#endif
		*p++ = 0;
		*p++ = 0;
		--len;
	}
#ifdef USE_SDL2
	if (underrun)
		SDL_AtomicAdd(&audio_underruns, 1);
#endif
}
```

- [ ] **Step 3: Track minimum occupancy**

At the end of `plat_sound_occupancy()`, convert the buffered frame count to a
percentage and update the minimum only when the audio ring exists:

```c
buffered = buffered * 100 / audio.buf_len;
#ifdef USE_SDL2
	if (!sdl_video_profile.audio_occupancy_samples ||
	    buffered < sdl_video_profile.audio_occupancy_min)
		sdl_video_profile.audio_occupancy_min = buffered;
	sdl_video_profile.audio_occupancy_samples++;
#endif
return buffered;
```

After the conversion, `buffered` is a percentage rather than a frame count.

- [ ] **Step 4: Report and reset the counters**

In `plat_sdl_profile_frame()`, snapshot the atomic counter before logging and
subtract exactly that snapshot so an interrupting callback increment is
preserved:

```c
int underruns = SDL_AtomicGet(&audio_underruns);
unsigned occupancy_min = sdl_video_profile.audio_occupancy_samples
	? sdl_video_profile.audio_occupancy_min : 0;

if (underruns)
	SDL_AtomicAdd(&audio_underruns, -underruns);
```

Extend the second profile line:

```c
PA_INFO("PROFILE sdl2: pace=%.2f/%u audio_wait_frame=%.2f/%u underruns=%d occupancy_min=%u%% ms avg/max\n",
	/* existing four timing arguments */,
	underruns,
	occupancy_min);
```

The existing `memset()` resets occupancy state after each interval.

- [ ] **Step 2: Return renderer readback status**

Change `plat_sdl_readback_screen()` from `void` to `int`:

```c
static int plat_sdl_readback_screen(void)
{
	uint64_t start_us;
	int ret;

	if (!renderer)
		return -1;

	start_us = plat_get_ticks_us_u64();
	ret = SDL_RenderReadPixels(renderer, NULL, SDL_PIXELFORMAT_RGB565,
				   screen_pixels, SCREEN_PITCH);
	if (ret != 0)
		PA_WARN("%s, SDL_RenderReadPixels failed: %s\n",
			__func__, SDL_GetError());
	plat_sdl_profile_add(&sdl_video_profile.readback_us,
			     &sdl_video_profile.readback_max_us,
			     plat_get_ticks_us_u64() - start_us);
	return ret;
}
```

Existing screenshot use may ignore the return value.
